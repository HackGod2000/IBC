## Getting Started

Clone the repository and run the below steps.

### Prerequisites

* node.js
* npm

### Installation

Open the local copy in VS Code.

- Hosting Backend 

```bash
cd Backend
npm install
npm start
```
- Serving Frontend

In Frontend folder,right click on index.html and open it with live server. [Live server is a extension on VS Code]
